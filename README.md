## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* pytube
* pydub
* ffmpeg


### Installation

1. Clone the repo
```sh
git clone https://github.com/samouei/behavioral-phenotyping.git
```
2. Install packages
```sh
pip install pytube
```
```sh
pip install pydub
```
```sh
brew install ffmpeg
```
